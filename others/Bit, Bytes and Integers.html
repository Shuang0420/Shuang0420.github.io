Bit-level manipulations

And

A = 1 and B = 1
=> A & B = 1

Or

A = 1 or B = 1
=> A | B = 1

not

A = 0
=> ~A = 1

Exclusive-Or(XOR)

either A=1 or B=1, but not both
=> A^B = 1
